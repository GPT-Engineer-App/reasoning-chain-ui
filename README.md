# reasoning-chain-ui

I want a UI that shows editable textboxes that are a chain of LLM output

Each textbox instructs what the future textboxes should say

The goal: make it easy to find errors in the reasoning relative to what I want to have

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/reasoning-chain-ui.git
cd reasoning-chain-ui
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
